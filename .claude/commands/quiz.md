현재 IDE에서 열려 있는 notes.md 파일의 내용을 기반으로 Quiz 섹션을 생성하여 파일 최하단에 추가하세요.

## Quiz 생성 규칙

1. 노트 내용을 먼저 Read 도구로 읽어 핵심 개념을 파악합니다.
2. 개념 확인, 코드 기반, 심화 문제를 골고루 섞어 3~5개의 퀴즈를 생성합니다.
3. 이미 Quiz 섹션이 있으면 기존 퀴즈 뒤에 이어서 추가합니다.

## Quiz 형식

- 섹션 제목: `## Quiz` (이미 있으면 생략)
- 질문: `**Q{번호}.** 질문 내용`
- 보기: `a) 보기1\` (줄 끝에 `\`로 줄바꿈 처리, 마지막 보기에는 `\` 없음)
- 코드가 필요하면 질문과 보기 사이에 코드 블록 삽입
- 정답: `<details><summary>정답 보기</summary>` 토글 안에 배치
- 정답 형식: `**정답)** — 간결한 해설 1~2문장`
- 퀴즈 사이에 `---` 구분선

## 형식 예시

```markdown
**Q1.** 질문 내용?

a) 보기1\
b) 보기2\
c) 보기3\
d) 보기4

<details>
<summary>정답 보기</summary>

**c) 보기3** — 해설 내용.

</details>

---

**Q2.** 다음 코드에서 어떻게 동작하는가?

\```html
<div style="display: flex;">
  <p>A</p>
</div>
\```

a) 보기1\
b) 보기2

<details>
<summary>정답 보기</summary>

**a) 보기1** — 해설 내용.

</details>
```

## 주의사항

- 노트에 없는 내용으로 퀴즈를 만들지 않습니다.
- 기술 용어는 노트와 동일한 표기를 사용합니다.
- 보기가 있는 문제와 단답형 문제를 섞습니다.
- 오답 보기도 그럴듯하게 만들어 변별력을 높입니다.
