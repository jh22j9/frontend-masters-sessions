사용자가 제공한 한국어 번역 스크립트를 기반으로 마크다운 노트 파일을 생성하세요.

## 파일 생성 규칙
1. 사용자가 지정한 강좌 디렉토리에 저장합니다.
2. 파일명은 `{챕터번호}-{주제-kebab-case}.md` 형식입니다 (예: `04-performance-benefits.md`).
3. 챕터 번호와 주제는 스크립트 내용에서 파악하거나 사용자에게 확인합니다.

## 노트 형식 규칙
- 제목: `# {강좌명} - {주제}` (H1)
- 본문은 `## 1. 섹션명`, `## 2. 섹션명` 형태로 번호를 매긴 섹션으로 구성합니다.
- 기술 용어는 한국어와 영어를 병기합니다 (예: 콜 스택(Call Stack))
- 핵심 개념은 **볼드**로 강조합니다.
- 코드 예시가 있으면 적절한 언어 태그와 함께 코드 블록으로 포함합니다.
- 관련 내용은 표(table)로 정리합니다.
- 구어체는 문어체로 다듬습니다.
- 섹션 사이에 `---` 구분선을 넣습니다.
- 스크립트에서 Q&A나 부연 설명이 있으면 `<details><summary>` 태그로 접을 수 있게 합니다.
- 불필요한 반복이나 잡담은 제거하고 핵심 내용만 정리합니다.

## 이미지
- 스크립트에서 시각 자료를 언급하면 `![설명](./images/{챕터번호}-{설명-kebab-case}.png)` 형식으로 이미지 참조를 남깁니다.
- images 폴더가 없으면 생성합니다.
