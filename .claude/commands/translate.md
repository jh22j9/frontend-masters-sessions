사용자가 제공한 영어 스크립트를 한국어로 번역하고, 번역 결과를 마크다운 파일로 생성해주세요.

규칙:
- 자연스러운 한국어로 번역하되, 기술 용어는 영어 원문을 병기하세요 (예: 서식 맥락(Formatting Context))
- 스크립트의 구어체는 문어체로 다듬어 번역하세요
- 원문의 단락 구분을 유지하세요
- 번역이 완료되면, 현재 IDE에서 열려 있는 transcript.md 파일에 직접 작성하세요
- 별도 파일을 새로 생성하지 마세요
- 파일 구조는 영어 원문 전체를 먼저 배치하고, `---` 구분선 아래에 한국어 번역본 전체를 배치하세요:
  ```
  ## 원문 (English)

  원문 전체 내용...

  ---

  ## 번역 (Korean)

  번역본 전체 내용...
  ```
- 원문의 모든 내용을 빠짐없이 번역하세요
