현재 대화 세션에서 바로 직전에 사용자가 한 질문과 그에 대한 답변을 추출하여, IDE에서 열려 있는 문서의 Q&A 섹션에 추가하세요.

## 질문·답변 추출 규칙

1. 현재 대화 히스토리를 역순으로 훑어 이 스킬 호출 직전의 사용자 메시지를 찾습니다.
2. 해당 사용자 메시지가 **질문**인지 확인합니다 (물음표 유무와 무관하게 궁금증·의문을 담은 메시지).
3. 그 질문에 대한 Claude의 **직전 응답**을 답변으로 사용합니다.
4. 질문이나 답변이 없으면 사용자에게 알립니다.

## 파일 탐색 규칙

1. IDE에서 현재 열려 있는 파일의 경로를 사용합니다.
2. 해당 파일을 Read 도구로 읽습니다.
3. `## Q&A` 섹션이 없으면 파일 최하단에 `## Q&A` 헤더를 먼저 추가합니다.

## Q&A 항목 추가 규칙

1. 기존 `## Q&A` 섹션 **최하단**에 새 항목을 추가합니다.
2. 항목 형식은 `<details><summary>` 토글입니다.
3. 질문 텍스트는 `<summary>` 안에 넣되, 사용자의 원문을 그대로 옮기지 않고 노트 문서에 어울리는 **명확하고 간결한 질문**으로 다듬습니다 (구어체 → 문어체, 핵심 의도 유지).
4. 답변은 핵심만 간결하게 다듬되, 중요한 예시·맥락은 유지합니다. 문체는 `~다` 대신 `~입니다/~습니다` 체로 작성합니다.
5. 기존 항목과 구분하기 위해 앞에 빈 줄을 넣습니다.

## 출력 형식

```markdown
<details>
<summary>질문 내용</summary>

답변 내용 (마크다운 허용)

</details>
```

## 주의사항

- 질문·답변을 임의로 지어내지 않습니다.
- 답변이 길면 핵심 내용 위주로 요약하되 기술 용어는 정확하게 유지합니다.
- 파일을 덮어쓰지 않고 Edit 도구로 Q&A 섹션에만 추가합니다.
